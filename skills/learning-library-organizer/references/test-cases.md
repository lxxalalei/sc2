# 学习资料库归档测试样例

## 样例 1：元数据完整的教材 PDF

输入：

- `downloaded_files[]` 包含 `stage=小学`、`grade=三年级`、`subject=数学`、`version=人教版`、`resource_type=教材`。

期望：

- 文件进入 `学习资料库/小学/三年级/数学/教材/人教版/`。
- 输出 `needs_review=false`。
- 最终资料库只出现 PDF 文件。

## 样例 2：主题资料但无年级

输入：

- 标题为“四则混合运算练习题”。
- `subject=数学`、`core_topic=四则混合运算`、`resource_type=习题`。

期望：

- 文件进入 `学习资料库/未分学段/未分年级/数学/四则混合运算/来源/`。
- 如果置信度达到阈值，`needs_review=false`。

## 样例 3：元数据很少

输入：

- 只有本地文件和标题，没有阶段、学科、主题、来源。

期望：

- 文件进入 `学习资料库/待确认/格式/`。
- 输出 `needs_review=true`。

## 样例 4：非法或缺失文件

输入：

- `local_file` 不存在。

期望：

- 不创建资料库文件。
- 写入 `failures`。
