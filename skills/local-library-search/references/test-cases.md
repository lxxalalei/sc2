# 本地资料库检索测试样例

## 样例 1：主题练习题命中

输入：

- 查询“四则混合运算练习题”。
- 索引中存在标题或主题为“四则混合运算”的 PDF 习题。

期望：

- 输出 1 个或多个 `learning-resource-candidate/v1` 候选。
- 候选包含 `local_file`。
- `raw.match_reasons` 包含主题或查询词命中。

## 样例 2：格式偏好过滤

输入：

- 查询“唐诗启蒙音频”。
- `format_preferences=["audio"]`。

期望：

- 音频资源优先。
- 非音频资源如果其他字段强匹配可以保留低分候选，但排序靠后。

## 样例 3：本地文件缺失

输入：

- 索引记录存在，但 `library_file` 文件已被删除。

期望：

- 默认跳过。
- 使用 `--include-missing` 时输出候选并标记“本地文件不存在”。

## 样例 4：无命中

输入：

- 查询“恐龙百科视频”，索引中没有相关资源。

期望：

- 输出空 `candidates`。
- 后续 flow 应继续调用其他 source skills，而不是结束流程。
