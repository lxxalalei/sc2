# 通用网页搜索测试样例

## 样例 1：可打印数学练习

输入查询和 agent 搜索结果：

```text
8岁 四则混合运算 练习题 可打印 PDF
```

期望：

- PDF/DOC/DOCX 链接识别为 `downloadable=true`。
- 资源类型优先识别为 `习题`。
- `subject` 或 `learning_domain` 识别为 `数学`。

## 样例 2：恐龙百科视频

输入查询：

```text
7岁 恐龙百科 视频 儿童
```

期望：

- 视频站或视频页面识别为 `resource_type=视频`。
- `topic` 识别为 `恐龙百科` 或 `恐龙`。
- 不直接下载视频。

## 样例 3：唐诗宋词启蒙音频

输入查询：

```text
5岁 唐诗宋词启蒙 音频 儿童
```

期望：

- 音频或朗读页面识别为 `resource_type=音频`。
- 成人赏析、考试资料不应被标为高置信度儿童资源。

## 样例 4：PPT 课件

输入查询：

```text
小学 三年级 分数初步认识 课件 PPT
```

期望：

- PPT/PPTX 链接识别为 `format=ppt` 或 `pptx`。
- 资源类型识别为 `课件`。

## 样例 5：广告或下载器风险

搜索结果标题包含：

```text
高速下载器 免费破解 学习资料合集
```

期望：

- `raw.warnings` 包含风险提示。
- 候选仍可输出，但交给 ranker 后应明显降权。
