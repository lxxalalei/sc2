# 学习资源评分测试样例

这些样例用于校验 `learning-resource-ranker` 是否能稳定排序并给出可解释理由。

## 样例 1：官方教材精确匹配

需求：下载人教版小学三年级数学上册教材。

期望：

- 国家平台官方教材排在前面。
- 相关性、权威性、元数据完整度高。
- 如 `requires_auth=true`，保留提示但不直接降为低质。

## 样例 2：8 岁四则混合运算练习题

需求：可打印练习题。

期望：

- PDF/DOC 习题优先。
- 视频讲解可以作为备选，但不应排在可打印练习题前。
- 年级或年龄明显不匹配的资源降权。

## 样例 3：恐龙百科视频

需求：7 岁孩子看的恐龙百科视频。

期望：

- 儿童向恐龙百科视频优先。
- 成人纪录片、标题夸张或来源不明的视频降权。
- 只有图文没有视频时，格式适配分较低。

## 样例 4：唐诗宋词启蒙音频

需求：5 岁孩子听赏。

期望：

- 儿童朗读、音频、可听赏资源优先。
- 只提供原文 PDF 的资源不应高分。
- 成人文学赏析、考研资料等降权。

## 样例 5：本地资料库查询

需求：看看本地有没有适合 6 岁孩子的识字资料。

期望：

- 本地已有资源优先级高。
- 低置信度分类进入备选，不直接强推荐。
- 如果没有来源 URL，不应仅因此拒绝本地资源。
